# milesng.me

## Deployment

This site is deployed automatically via [Cloudflare Pages](https://pages.cloudflare.com/) using the source code from this GitHub repository.

### How it works

1. Push changes to the `main` branch on GitHub.
2. Cloudflare Pages detects the update and triggers a new deployment.
3. The site is built and published at your Cloudflare Pages URL.
